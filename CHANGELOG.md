### Added

- Initial release
- Parse BNP Paribas bank statements (PDF) and extract operations to YAML
- CLI tools: `my-bank-statements-parser`, `my-bank-statements-ventilate`, `my-bank-statements-add-patterns`
- Ventilation module for categorizing operations into budget categories
- GitHub Actions CI and publish workflows
